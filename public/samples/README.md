# Sample Files

This directory contains placeholder sample files for the demo.
Replace these with real files for production demo use.

## Required sample files

| File | Demo | Notes |
|------|------|-------|
| `invoice-sample.pdf` | Invoice Extraction | A sample invoice PDF with vendor, line items, totals |
| `mixed-documents.pdf` | Classify & Split | A multi-page PDF with mixed document types |
| `chart-sample.png` | Image Summary | A bar/line chart or dashboard screenshot |
| `call-recording.wav` | Audio Analysis | A short customer service call recording |
| `video-clip.mp4` | Video Analysis | A 1-2 minute explainer or demo video |
| `service-request-form.pdf` | Custom Form | A service request or intake form |
| `article-sample.pdf` | Markdown Conversion | A multi-page article or report PDF |
| `insurance-claim.pdf` | Extraction w/ Review | An insurance claim form with handwritten/typed fields |

## How to add samples

1. Place real files in this directory matching the names above.
2. The demo app serves these files under `/samples/` so the "Use Sample File" button works.
3. If no sample is available, users can still drag-and-drop or browse for their own files.

## Placeholder files

The `.txt` placeholder files in this directory are stand-ins so the project structure is complete.
They will not work as actual sample inputs — replace them with real documents/media files.
